%   - Standard errors in parentheses

%   - {{TITLE}}          : Table title

%   - {{COEF_*}}         : Coefficient estimates
%   - {{SE_*}}           : Standard errors
%   - {{STARS_*}}        : Significance stars (*, **, ***)
%   - {{N_*}}            : Sample sizes
%   - {{R2_*}}           : R-squared values
%   - {{FE_*}}           : Fixed effects indicators (Yes/No)
% =============================================================================
