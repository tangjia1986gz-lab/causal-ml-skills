% USAGE NOTES:
% =============================================================================
%
% 1. siunitx Column Formats:
%    - S[table-format=5.0]   : 5-digit integers (e.g., N)
%    - S[table-format=2.3]   : 2 digits before decimal, 3 after
%    - S[table-format=-1.3]  : Allow negative sign, 1 digit, 3 decimals
%    - Wrap text in braces {} when using S columns
%
% 2. Balance Test Interpretation:
%    - p > 0.10: Groups are balanced on this covariate
%    - Joint F-test: Tests overall balance across all covariates
%    - Normalized difference < 0.25: Imbens & Rubin (2015) threshold
%
% 3. For RCT balance tables:
%    - Add stratification variables if randomization was stratified
%    - Report F-test for joint balance
%    - Consider showing baseline outcome if available
%
% 4. For observational studies:
%    - Show balance before and after matching/weighting
%    - Use standardized mean differences
%    - Consider variance ratios for continuous variables
%

%    - Replace {{PLACEHOLDER}} with actual values
