% USAGE NOTES:
% =============================================================================
%
% 1. Reference Period Selection:
%    - Standard: t=-1 (last pre-treatment period)
%    - Alternative: t=-2 for extra margin before treatment
%    - Never use t=0 as reference (post-treatment)
%
% 2. Endpoint Binning:
%    - Bin extreme periods to avoid sparse cells
%    - Common choices: bin at -4/-5 and +4/+5
%    - Report number of observations in binned categories
%
% 3. Pre-Trends Testing:
%    - Joint F-test: H0: all pre-treatment coefficients = 0
%    - Report both test statistic and p-value
%    - Failing to reject is necessary but not sufficient for parallel trends
%
% 4. Staggered DID Considerations:
%    - TWFE can be biased with heterogeneous treatment effects
%    - Use Sun-Abraham, Callaway-Sant'Anna, or Borusyak et al.
%    - Report multiple estimators for robustness
%    - Always report number of treatment cohorts
%
% 5. Estimation Details to Report:
%    - Clustering level for standard errors
%    - Fixed effects included (unit, time, unit-trends)
%    - Control variables if any
%    - Sample restrictions
%
% 6. Software References:
%    - Python: linearmodels, pyfixest, differences
%    - R: fixest, did, didimputation
%    - Stata: eventdd, csdid, did_imputation
%
% 7. Publication Tips:
%    - Always pair with event study figure
%    - Report both individual period effects and overall ATT
%    - Discuss any notable pre-trend coefficients
%    - Compare across robust estimators if staggered design
% =============================================================================
