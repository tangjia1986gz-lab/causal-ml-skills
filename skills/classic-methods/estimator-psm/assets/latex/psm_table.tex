\end{tabular}
\begin{tablenotes}
\footnotesize
\item \textit{Notes:} $\Gamma$ represents the odds ratio by which an unobserved confounder
could alter the odds of treatment.
\item $\Gamma = 1$: No hidden bias (unconfoundedness assumption holds).
\item $\Gamma = 2$: Unobserved confounder could double the odds of treatment.
\item Critical $\Gamma$: The point at which the upper-bound $p$-value exceeds 0.05.
\item \textit{Interpretation:} Results are robust to unobserved confounders that change
treatment odds by up to [PLACEHOLDER: critical gamma]-fold.
\end{tablenotes}
