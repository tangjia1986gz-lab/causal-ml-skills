# Pre-Submission Checklist for Economics Papers

Use this checklist before submitting to journals or posting working papers.

---

## Content Checks

### Abstract
- [ ] Contains motivation (why this matters)
- [ ] States research question clearly
- [ ] Mentions identification strategy
- [ ] Includes main quantitative finding with standard error/CI
- [ ] States implication/contribution
- [ ] Within word limit (check target journal)
- [ ] JEL codes included (if required)
- [ ] Keywords included (3-5)

### Introduction
- [ ] Opens with hook/motivation
- [ ] States what this paper does
- [ ] Previews main finding with numbers
- [ ] Clearly states contribution
- [ ] Positions paper in literature (briefly)
- [ ] Includes roadmap paragraph
- [ ] Reader knows the punchline by page 2

### Literature Review
- [ ] Focuses on directly relevant papers
- [ ] Explains relationship of each cited paper to yours
- [ ] Identifies gap you fill
- [ ] Fair to prior work
- [ ] Includes recent papers (last 2-3 years)
- [ ] Not just a list of summaries

### Data/Background
- [ ] Data source clearly described
- [ ] Sample construction explained
- [ ] Key variables defined
- [ ] Summary statistics table included
- [ ] Data limitations acknowledged
- [ ] Institutional context sufficient for identification

### Methodology
- [ ] Identification strategy clearly explained
- [ ] Identifying assumption stated explicitly
- [ ] Estimating equation provided with all terms defined
- [ ] Standard error clustering justified
- [ ] Threats to identification discussed
- [ ] Connection between variation and causal interpretation clear

### Results
- [ ] Main finding stated with magnitude interpretation
- [ ] Economic significance discussed (not just statistical)
- [ ] Tables walk reader through specifications
- [ ] Robustness checks included
- [ ] Heterogeneity explored (if applicable)
- [ ] Results consistent with identification story

### Conclusion
- [ ] Summarizes main finding
- [ ] Discusses policy/theoretical implications
- [ ] Acknowledges limitations honestly
- [ ] Suggests future research
- [ ] Does not introduce new results

---

## Formatting Checks

### Tables
- [ ] Three-line format (top, header, bottom rules only)
- [ ] No vertical lines
- [ ] Standard errors in parentheses
- [ ] Stars defined (*** p<0.01, ** p<0.05, * p<0.1)
- [ ] Clustering explained in notes
- [ ] Dependent variable stated
- [ ] Sample size included
- [ ] Variable labels (not Stata/R names)
- [ ] Consistent decimal places (usually 2-3)
- [ ] Referenced in text before appearing

### Figures
- [ ] Axis labels with units
- [ ] Legend (if multiple series)
- [ ] Confidence intervals shown
- [ ] Source noted
- [ ] Readable at journal print size
- [ ] Works in black and white
- [ ] High resolution (300+ dpi for raster)
- [ ] Referenced in text before appearing

### Equations
- [ ] Numbered if referenced
- [ ] All notation defined
- [ ] Consistent notation throughout
- [ ] Greek letters match across text, tables, figures

### Citations
- [ ] All claims have citations
- [ ] All citations in reference list
- [ ] All references cited in text
- [ ] Consistent format throughout
- [ ] Working papers updated to published versions
- [ ] DOIs included where available

---

## Writing Quality Checks

### Clarity
- [ ] Main point of each paragraph clear
- [ ] Active voice predominates
- [ ] Jargon explained when first used
- [ ] Sentences are concise
- [ ] No wall-of-text paragraphs

### Precision
- [ ] Numbers include uncertainty measures
- [ ] Claims match evidence strength
- [ ] Causal language appropriate for design
- [ ] Statistical vs. economic significance distinguished

### Flow
- [ ] Sections follow logical order
- [ ] Transitions between sections
- [ ] Each table/figure has a clear purpose
- [ ] No redundant sections

---

## Technical Checks

### Document
- [ ] Title page complete (authors, affiliations, acknowledgments)
- [ ] Page numbers on all pages
- [ ] Line numbers (if required for submission)
- [ ] Double-spaced (if required)
- [ ] Margins appropriate
- [ ] Font readable (12pt typically)

### References
- [ ] Alphabetized correctly
- [ ] Format matches journal style
- [ ] All author names spelled correctly
- [ ] Publication years verified
- [ ] Journal names correct

### Appendix
- [ ] Clearly labeled (Appendix A, B, etc.)
- [ ] Tables/figures numbered separately (A1, A2, etc.)
- [ ] Referenced in main text
- [ ] Includes replication materials notice

### Data/Code
- [ ] Replication package prepared
- [ ] README file written
- [ ] Code commented and organized
- [ ] Data files or access instructions included
- [ ] Output matches paper exactly

---

## Journal-Specific Checks

### Before Choosing Journal
- [ ] Paper fits journal scope
- [ ] Quality appropriate for journal tier
- [ ] Length within typical range
- [ ] Not previously rejected from this journal (check rules)

### Submission Preparation
- [ ] Read author guidelines completely
- [ ] Word/page limit met
- [ ] Abstract within limit
- [ ] Required sections included
- [ ] File format correct (PDF, Word, LaTeX)
- [ ] Blinded version prepared (if required)
- [ ] Cover letter drafted

---

## Final Review

### Self-Check
- [ ] Read paper aloud (catches awkward phrasing)
- [ ] Print and read on paper (catches formatting issues)
- [ ] Check all numbers match between text and tables
- [ ] Verify figure/table references are correct
- [ ] Run spell-check one final time

### External Check
- [ ] At least one colleague has read and commented
- [ ] Addressed all substantive feedback
- [ ] Presentation at seminar/conference (if possible)
- [ ] Junior colleagues can follow the argument

---

## Common Last-Minute Issues

### Easy to Miss
- [ ] Acknowledgments include everyone who helped
- [ ] Funding sources disclosed
- [ ] Conflicts of interest declared (if any)
- [ ] Data availability statement included
- [ ] IRB/ethics approval mentioned (if human subjects)

### Embarrassing If Caught
- [ ] Author names spelled correctly
- [ ] Affiliation current
- [ ] Email addresses work
- [ ] Title consistent across documents
- [ ] Abstract matches paper content

---

## Submission Checklist

- [ ] Main manuscript PDF
- [ ] Separate tables/figures (if required)
- [ ] Online appendix
- [ ] Cover letter
- [ ] Author information form
- [ ] Suggested referees (3-5 names)
- [ ] Referees to exclude (if any)
- [ ] Replication package (or commitment to provide)
- [ ] Previous submission history (if requested)

---

## After Submission

- [ ] Save confirmation email
- [ ] Note submission date
- [ ] Set calendar reminder to check status
- [ ] Continue working on paper (don't wait for decision)
- [ ] Prepare presentation version
